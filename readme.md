# Brief description

User-friendly platform for primary data analysis, distribution identification, histogram generation, ECDF, and KDE plotting

## Main Technologies
1. **Server**: 
   - Node.js
   - TypeScript
   - Express
2. **Web**: 
   - React.js
   - react-router-dom
   - recharts
   - html2canvas

## Usage

1. Start the server:
    ```sh
    cd ./server
    npm start
    ```

2. Start the web application:
    ```sh
    cd ./web
    npm start
    ```
